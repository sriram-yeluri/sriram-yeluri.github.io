Install virtualbox Extension pack

Link

1. wget http://download.virtualbox.org/virtualbox/5.2.4/Oracle_VM_VirtualBox_Extension_Pack-5.2.4-119785.vbox-extpack 
2. sudo vboxmanage extpack Oracle_VM_VirtualBox_Extension_Pack-5.2.4-119785.vbox-extpack
3. Once installed, you can see at : File -> Preferences -> Extensions
4. To verify if it has been successfully installed, we may list all installed extension packs:
VBoxManage list extpacks
